this project is referencing three external python libraries, which are pygame, pybox2d(box2d), python image library(PIL).
the libraries and the main program should all be running under python 3.6.2 environment
make sure the libraries are all properly installed before launching

all the libraries could be downloaded through this link (if using windows10 system)(using pip):

	https://www.lfd.uci.edu/~gohlke/pythonlibs/
